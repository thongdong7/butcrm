# Development

For mac:

1. Install xcode
2. Install git
3. Run `sudo git` to accept the agreement
4. Clone this project

```
git clone git@github.com:thongdong7/butcrm.git
```

5. Install homebrew
6. Install npm

```
brew install npm
```

7. Install node packages

```
npm install
```

8. Run

```
react-native start
```

```
react-native run-android
```
